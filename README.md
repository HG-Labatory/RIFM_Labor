# Pygame Project

Ein einfaches Pygame-Projekt für Spieleentwicklung.

## Voraussetzungen

- Python 3.8 bis 3.12 (Pygame 2.6.1 unterstützt Python 3.14 noch nicht)

## Installation

1. Stelle sicher, dass Python 3.11 installiert ist.
2. Erstelle eine virtuelle Umgebung: `python -m venv venv311` (verwende Python 3.11)
3. Aktiviere die Umgebung: `venv311\Scripts\activate` (Windows)
4. Installiere die Abhängigkeiten: `pip install -r requirements.txt`

## Ausführen

Führe `python main.py` aus, um das Spiel zu starten.

## Projektstruktur

- `main.py`: Hauptdatei des Spiels
- `assets/`: Ordner für Bilder, Sounds usw.
- `requirements.txt`: Abhängigkeiten
- `.gitignore`: Ignorierte Dateien für Git
